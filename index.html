MIDITOOLS
A set of free browser-based tools for MIDI musicians. No install. No account. Works on Android Chrome over USB.
Requires Chrome for Android — Web MIDI API is supported in Chrome on Android.
When prompted, allow MIDI access. No special setup needed — these tools run over HTTPS so Chrome handles permissions automatically.

About These Tools

Built for solo loopers and live performers working with the Boss RC-600 Loop Station. These tools grew out of a real setup process — figuring out MIDI routing, ASSIGN mapping, and Cubase integration from scratch.

All tools run entirely in your browser. No data is sent anywhere. The planner saves to your device's local browser storage only. Source available at github.com/adebyrne.
