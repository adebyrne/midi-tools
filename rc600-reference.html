Boss RC-600 MIDI Implementation

The RC-600 does not have fixed CC→function mappings. It uses a flexible ASSIGN system — you pick any valid CC# and map it to any target function. Fixed messages (Program Change, Clock, Start/Stop) always work with no setup. Valid CC numbers for ASSIGN are CC#1–31 and CC#64–95.

How to Set Up an Assignment on RC-600
StepAction
1Press MENU on the RC-600
2Navigate to ASSIGN
3Select any of the 16 available ASSIGN slots (1–16)
4Set SOURCE to MIDI CC# and choose your CC number
5Set TARGET to the function you want (e.g. TRK4 REC/PLY)
6Set SW MODE to TOGGLE (latching) or MOMENT (held)
7Press EXIT + ENTER simultaneously to WRITE/SAVE to memory
⚠Assigns are per-memory, not global. Save to every memory that needs them.
