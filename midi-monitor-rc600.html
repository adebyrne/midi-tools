Boss RC-600 MIDI Reference

The RC-600 does not have fixed CC→function mappings. Instead it uses a flexible ASSIGN system — you pick any CC# and map it to any target function on the device. The table below shows what functions are available as targets, and what fixed messages (Program Change, Clock, Start/Stop) always work.
⌕
Fixed Messages — Always Active
MIDI MsgValueRC-600 FunctionNotes
Program Change 1–99 Switch Memory 01–99 FIXED PC 1 → Memory 01, PC 99 → Memory 99. Messages 100–128 ignored.
MIDI Start (FA) — All Tracks Start FIXED All tracks begin playback simultaneously. Set SYNC CLOCK to AUTO.
MIDI Stop (FC) — All Tracks Stop FIXED Use ASSIGN with SOURCE=SYNC ST/STP and TARGET=ALL ST/STP to enable.
MIDI Clock (F8) — Tempo Sync FIXED RC-600 syncs BPM to incoming clock. Set SYNC CLOCK → AUTO.
Bank Select CC#0 / CC#32 any Ignored IGNORED Bank Select messages are discarded by the RC-600.
Assignable CC Targets (MENU → ASSIGN)
ASSIGN TargetCC# RangeValueWhat it Does
TRK1–6 REC/PLY ASSIGN CC#1–31, 64–95 0–127 Triggers Record → Play → Overdub cycle for that track. Most common MIDI control target.
TRK1–6 STOP ASSIGN CC#1–31, 64–95 0–127 Stops playback/recording on the specified track.
TRK1–6 CLEAR ASSIGN CC#1–31, 64–95 0–127 Clears (erases) the loop on the specified track.
TRK1–6 UNDO/REDO ASSIGN CC#1–31, 64–95 0–127 Undoes or redoes the last overdub on that track.
TRK1–6 PLAY LEVEL ASSIGN CC#1–31, 64–95 0–200 Controls the playback volume of that track (expression pedal friendly).
TRK1–6 REVERSE ASSIGN CC#1–31, 64–95 OFF/ON Toggles reverse playback on/off for the track.
TRK1–6 1SHOT ASSIGN CC#1–31, 64–95 OFF/ON Toggles one-shot (play once and stop) mode for the track.
ALL ST/STP ASSIGN CC#1–31, 64–95 0–127 Starts or stops all tracks simultaneously.
RHYTHM ST/STP ASSIGN CC#1–31, 64–95 0–127 Starts or stops the onboard rhythm/drum pattern.
RHYTHM VOLUME ASSIGN CC#1–31, 64–95 0–200 Controls the rhythm output volume.
RHYTHM KIT ASSIGN CC#1–31, 64–95 STUDIO→808+909 Switches between 16 drum kits (Studio, Live, Rock, Jazz, 808+909, etc.).
RHYTHM VARIATION ASSIGN CC#1–31, 64–95 A–D Switches between the 4 variations (A/B/C/D) of the current rhythm pattern.
INPUT FX A–D SW ASSIGN CC#1–31, 64–95 OFF/ON Turns Input FX A, B, C, or D on or off.
TRACK FX A–D SW ASSIGN CC#1–31, 64–95 OFF/ON Turns Track FX A, B, C, or D on or off.
INPUT FX BANK SW ASSIGN CC#1–31, 64–95 OFF/ON Turns the entire Input FX bank (A–D) on or off.
TRACK FX BANK SW ASSIGN CC#1–31, 64–95 OFF/ON Turns the entire Track FX bank (A–D) on or off.
MASTER LEVEL ASSIGN CC#1–31, 64–95 0–200 Controls the overall master output level.
TEMPO ASSIGN CC#1–31, 64–95 40–250 BPM Sets the loop/rhythm tempo.
MEMORY INC/DEC ASSIGN CC#1–31, 64–95 0–127 Steps through memories up or down by one.
CURRENT TRACK ASSIGN CC#1–31, 64–95 TRK1–6 Sets which track is the active/selected edit target.
How to Set Up an Assignment on RC-600
StepAction
1Press MENU on the RC-600
2Navigate to ASSIGN
3Select any of the 16 available ASSIGN slots (1–16)
4Set SOURCE to MIDI CC# and choose your CC number (e.g. CC#10)
5Set TARGET to the function you want (e.g. TRK1 REC/PLY)
6Set SW MODE to TOGGLE (latching) or MOMENT (held)
7Press EXIT + ENTER simultaneously to WRITE/SAVE to memory
⚠Assigns are per-memory, not global. You must save to every memory that needs them.
Valid CC Numbers for RC-600 ASSIGN
RangeCC Numbers
Lower Range CC#1 – CC#31 (31 numbers)
Upper Range CC#64 – CC#95 (32 numbers)
Not supported CC#0 (Bank Select), CC#32–63, CC#96–127 are not usable as ASSIGN sources
